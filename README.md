# income-analysis-russia

Среднедушевые денежные доходы населения в регионах России

Синопсис
В данном проекте проводится анализ среднедушевых денежных доходов населения по регионам Российской Федерации. Цель исследования — выявить различия в уровне доходов между регионами, определить лидеров и аутсайдеров, а также оценить динамику изменения доходов во времени. Результаты анализа визуализированы с помощью диаграмм и графиков для наглядного представления данных.
Проект представляет собой миниатюрное data-driven исследовании, от сбора и очистки данных до анализа и визуализации. Основная задача была продемонстрировать навыки работы с данными и их интерпретации.

Актуальность
Доходы населения являются одним из ключевых показателей уровня жизни и экономического развития региона. Анализ доходов по субъектам РФ позволяет выявить региональное неравенство, определить социально-экономически развитые и отстающие территории, а также понять, какие регионы требуют особого внимания в плане экономической политики.

Исследовательские вопросы
Какие регионы имеют самые высокие среднедушевые доходы?
Какие регионы находятся в нижней части рейтинга доходов?
Как изменяются доходы населения по годам?
Насколько велика разница в доходах между регионами?

Данные получены из официального источника — Росстат https://rosstat.gov.ru 
Процесс подготовки данных включал:
очистку данных от пропусков и дубликатов,
приведение числовых значений к единому формату,
стандартизацию названий регионов.
Все исходные и обработанные данные сохранены в папке [[data/cleaned_data.csv](https://github.com/wygsaa/income-analysis-russia/edit/main/README.md#:~:text=cleaned_data.csv)

Анализ
В рамках проекта были выполнены следующие шаги анализа:
Расчёт основных описательных статистик: среднее, медиана, минимальное и максимальное значения доходов.
Определение лидеров и аутсайдеров по доходам (топ-10 и анти-топ-10 регионов).
Сравнение доходов по регионам с помощью сводных таблиц и расчет доли каждого региона в общей сумме доходов.
Визуализация данных через столбчатые, линейные и круговые диаграммы.
Применение условного форматирования для наглядного выделения высоких и низких значений.
